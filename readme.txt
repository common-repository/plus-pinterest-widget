== Description ==

Plus Pinterest Widget is amazing wordpress widget plugin to display pinterest images on your wordpress website. Enabling you to post a feed directly from Pinterest on your pages! The Pinterest Widget is easy to install, easy to use and provides users with an ample amount of benefits.

You can look at demo here: http://twitterslider.com/wp/

== Installation ==

Thanks for installing or downloading our Pinterest Widget

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder

2. Activate the plugin

3. You will see facebook settings from right side of your wordpress menu.

Plus Pinterest Widget Settings:

Pinterest Widget Allow You To Display Pinterest Image In Your Website,

- Add Widget Title           (Example = Plus Pinterest Widget)
- Add Pinterest user URL     (Example = https://www.pinterest.com/pinterest/) 
- Add Pinterest Board Width  (Example = 500)
- Add Pinterest Board Height (Example = 500)
- Add Pinterest Image Width  (Example = 120)
